# DevJam IX (Catch-Up)

[[toc]]

## Time and Duration

- Start Date: 2026-02-01
- Launch Date: 2026-04-30

## Point System

Max Earnable Points: 6, or the number of points you are away from 50, depending on which is smaller.

This table details earnable points for the Catch-Up DevJam:

| Points |                   Set Submission                               |
| :----: | :------------------------------------------------------------: |
|   4    |                Licensed Saturn RPGs                            |
|   3    |                Standard Apple II games                         |
|   3    |                Licensed MSX Games                              |
|   3    |                Standard PC-8801 games                          |
|   3    |                Licensed PC Engine (CD) Games                   |
|   3    |                Licensed Saturn Games                           |
|   3    |                Licensed Sega CD Games                          |
|   3    |                Atomiswave and Sega NAOMI games                 |
|   2    |                Other Licensed Games                            |
|   2    |                Homebrews, Hacks, etc.                          |
|   2    |                Collaborations (3-point games)                  |
|   1    |                Collaborations (2-point games)                  |
|   1    |                Late Submissions                                |

## Quarterly Goals

This DevJam, as a catch-up period, does not have any special goals.

The list of eligible platforms is as follows:

- Apple II
- Arcade
- MSX
- PC-8001/PC-8801
- PC Engine
- PC Engine CD
- Sega 32X
- Sega CD
- Sega Saturn
- Sega SG-1000
