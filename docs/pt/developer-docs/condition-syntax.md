# Sintaxe de Condição

### Tamanhos de memória

|                   Tamanho                    | Prefixo |  Exemplo   |
| :---------------------------------------: | :----: | :--------: |
|                   Bit0                    | `0xM`  | `0xM01234` |
|                   Bit1                    | `0xN`  | `0xN01234` |
|                   Bit2                    | `0xO`  | `0xO01234` |
|                   Bit3                    | `0xP`  | `0xP01234` |
|                   Bit4                    | `0xQ`  | `0xQ01234` |
|                   Bit5                    | `0xR`  | `0xR01234` |
|                   Bit6                    | `0xS`  | `0xS01234` |
|                   Bit7                    | `0xT`  | `0xT01234` |
|                  Lower4                   | `0xL`  | `0xL01234` |
|                  Upper4                   | `0xU`  | `0xU01234` |
|                   8bit                    | `0xH`  | `0xH01234` |
|                   16bit                   | `0x `  | `0x 01234` |
|                   24bit                   | `0xW`  | `0xW01234` |
|                   32bit                   | `0xX`  | `0xX01234` |
|                 16bit BE                  | `0xI`  | `0xI01234` |
|                 24bit BE                  | `0xJ`  | `0xJ01234` |
|                 32bit BE                  | `0xG`  | `0xG01234` |
| [BitCount](/pt/developer-docs/bitcount-size) | `0xK`  | `0xK01234` |
|                   Float                   |  `fF`  | `fF01234`  |
|                 Float BE                  |  `fB`  | `fB01234`  |
|                 Double32                  |  `fH`  | `fH01234`  |
|                Double32 BE                |  `fI`  | `fI01234`  |
|                   MBF32                   |  `fM`  | `fM01234`  |
|                 MBF32 LE                  |  `fL`  | `fL01234`  |
